# k.k 텔레그램 봇 만들기

- InlineKeyboard 기능을 포함한 기능을 만들었음.

- 문제점
  - 한번 ```/bye``` 메시지 전송하면 텔레그램봇이 stop()함수를 실행함.
  - stop()함수 실행 후에는 다시 시작할 수 없음.
  - 다시시작하는 기능을 추가
  
- 다음에 해볼 기능
  - restart() 기능 추가해야함.
  - 각 keyboard의 단추를 눌렀을 때 반응을 다르게 해야함.
  - Custom Keyboard 도 사용해보자.
